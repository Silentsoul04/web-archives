Sunnet 教育訓練管理系統 - Broken Access Control

TVN ID TVN-201910011
CVE ID CVE-2020-10510
CVSS 8.1 (High)
CVSS:3.1/AV:N/AC:L/PR:L/UI:N/S:U/C:H/I:H/A:N
影響產品 Sunnet教育訓練管理系統8、9版
問題描述 Sunnet教育訓練管理系統未恰當設定存取控制，攻擊者登入系統後，可透過特定網址執行未經授權的功能。
解決方法 升級到教育訓練管理系統10版以上，或洽旭聯科技取得修正版本
漏洞通報者 Jia-Rong Chen
公開日期 2020-03-27
