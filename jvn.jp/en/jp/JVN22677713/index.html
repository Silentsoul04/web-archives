JVN#22677713
OpenEMR vulnerable to authentication bypass

Overview

OpenEMR contains an authentication bypass vulnerability.

Products Affected

  • OpenEMR 2.8.3 to 4.2.0 patch 1

Description

OpenEMR is an electronic health records and medical practice management application. OpenEMR contains an authentication bypass vulnerability (CWE-302).

Impact

Sensitive information may be obtained by a remote attacker who can access the web interface of the product.

Solution

Update the software and apply the patch
This vulnerability has been addressed in OpenEMR 4.2.0 patch 2.
Update the software and then apply the patch according to the information provided by the developer.
