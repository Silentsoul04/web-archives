JVN#11622218
iChain Insurance Wallet App for iOS vulnerable to directory traversal

Overview

iChain Insurance Wallet App for iOS contains a directory traversal vulnerability.

Products Affected

  • iChain Insurance Wallet App for iOS Version 1.3.0 and earlier

Description

iChain Insurance Wallet App for iOS provided by iChain, Inc. uses the old version of cordova-plugin-ionic-webview, and inherits a directory traversal vulnerability (CWE-22, CVE-2018-16202).

Impact

A remote attacker may obtain an arbitrary file such as a file related to an application on iOS device. As a result, contents of the file may be leaked.

Solution

Update the Application
Update the application to the latest version according to the information provided by the developer.
