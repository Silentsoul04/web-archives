JVN#68536660
Archive Decoder may insecurely load executable files

Overview

Archive Decoder may use unsafe methods for determining how to load executables (.exe).

Products Affected

  • Archive Decoder Ver.1.23 and earlier

Description

Archive Decoder is a file extraction software that supports multiple file formats. Archive Decoder loads certain executables (.exe) when extracting files. Archive Decoder contains an issue with the file search path, which may insecurely load executables.

Impact

An attacker may execute arbitrary code with the privilege of running the application.

Solution

Update the Software
Update to the latest version according to the information provided by the developer.
