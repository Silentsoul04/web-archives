JVN#80583739
Cybozu Garoon vulnerable to cross-site scritping

Overview

Cybozu Garoon contains a cross-site scripting vulnerability in the function "Notices portlet".

Products Affected

  • Cybozu Garoon 2.0.0 to 3.7 Service Pack 3

Description

Cybozu Garoon provided by Cybozu, Inc. is a groupware. Cybozu Garoon contains an issue in the function "Notices portlet", which may result in a cross-site scripting vulnerability (CWE-79).

Impact

An arbitrary script may be executed on the web browser of a user that is logged on.

Solution

Update the Software
Update to the latest version according to the information provided by the developer.
