JVN#02037158
AttacheCase vulnerable to arbitrary script execution

Overview

AttacheCase contains an arbitrary script execution vulnerability.

Products Affected

  • AttacheCase ver.2.8.4.0 and earlier - CVE-2018-0674
  • AttacheCase ver.3.3.0.0 and earlier - CVE-2018-0675

Description

AttacheCase is an open source file encryption software provided by HiBARA Software.
If a setting file _AtcCase.ini is specially crafted and it resides in the same folder where ATC file resides, it is leveraged to execute an arbitrary script when ATC file is decrypted.

Impact

A remote unauthenticated attacker may execute an arbitrary script.

Solution

Update the software
Update to the latest version according to the information provided by the developer.
