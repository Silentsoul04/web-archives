JVN#16485017
SLFileManager for Android vulnerable to directory traversal

Overview

SLFileManager for Android provided by S-Link, Inc. contains a directory traversal vulnerability.

Products Affected

  • SLFileManager version 1.2.5 and earlier

Description

SLFileManager provided by S-Link, Inc. contains a flaw in processing file names, which may result in a directory traversal (CWE-22) vulnerability.

Impact

A remote, unauthenticated attacker may create an arbitrary file or overwrite an existing file in a directory that the application has privileges to access.

Solution

Apply an Update
Update to the latest version according to the information provided by the developer.
