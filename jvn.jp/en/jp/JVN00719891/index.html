JVN#00719891
Multiple vulnerabilities in CG-WLR300NM

Overview

CG-WLR300NM provided by Corega Inc. contains multiple vulnerabilities.

Products Affected

  • CG-WLR300NM firmware version 1.90 and earlier

Description

CG-WLR300NM provided by Corega Inc. is a wireless LAN router.  CG-WLR300NM contains multiple vulnerabilities listed below.

  • OS Command Injection (CWE-78) - CVE-2017-10813
    CVSS v3 CVSS:3.0/AV:A/AC:L/PR:H/UI:N/S:U/C:H/I:H/A:H Base Score: 6.8
    CVSS v2 AV:A/AC:L/Au:S/C:P/I:P/A:P Base Score: 5.2
  • Buffer Overflow (CWE-119) - CVE-2017-10814
    CVSS v3 CVSS:3.0/AV:A/AC:L/PR:H/UI:N/S:U/C:H/I:H/A:H Base Score: 6.8
    CVSS v2 AV:A/AC:L/Au:S/C:P/I:P/A:P Base Score: 5.2

Impact

  • A user who can access the administrative console of the device may execute an arbitrary OS command - CVE-2017-10813
  • A user who can access the administrative console of the device may execute arbitrary code - CVE-2017-10814

Solution

Do not use CG-WLR300NM
Stop using CG-WLR300NM.  According to the developer, there is no plan to provide fix for these vulnerabilities since CG-WLR300NM is no longer supported.
