JVN#09268287
Multiple Buffalo network devices vulnerable to cross-site request forgery

Overview

Multiple network devices provided by BUFFALO INC. contain a cross-site request forgery vulnerability.

Products Affected

  • WSR-1166DHP Ver.1.01 and earlier
  • WHR-1166DHP Ver.1.90 and earlier
  • WHR-600D Ver.1.90 and earlier
  • WHR-300HP2 Ver.1.90 and earlier
  • WMR-433 Ver.1.01 and earlier
  • WMR-300 Ver.1.90 and earlier
  • WEX-300 Ver.1.90 and earlier
  • BHR-4GRV2 Ver.1.04 and earlier

Description

Multiple network devices provided by BUFFALO INC. contain a cross-site request forgery vulnerability (CWE-352).

Impact

If a user views a malicious page while logged in, unintended operations may be conducted.

Solution

Update the firmware
Update the firmware according to the information provided by the developer.
