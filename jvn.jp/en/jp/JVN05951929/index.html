JVN#05951929
sp mode mail issue where emails in the process of creation may be accessed

Overview

sp mode mail provided by NTT DOCOMO contains an issue where emails may be accessed while in the process of being created.

Products Affected

  • sp mode mail rev.6100 through rev.6300 for Android 4.0.X and earlier
  • sp mode mail rev.6130 through rev.6700 for Android 4.1 and later

Description

sp mode mail provided by NTT DOCOMO contains an application link interface so that mail data can be exchanged with external application during email creation. When the application to be linked is selected, the email contents and attachment are saved to the SD card, therefore other Android applications may access this data.

Impact

If a malicious Android application is installed on the device, contents of emails being created may be obtained.

Solution

An update to address this issue will not be provided by the developer.

Note on usage:
According to the developer, a warning about this issue has been included in the consents agreement that is displayed when launching the application for the first time, the pop-up when using this application link interface for the first time and the help page for this product in sp mode mail rev.6400 and later for Android 4.0.X and earlier and sp mode mail rev.6800 and later for Android 4.1 and later.
