JVN#82465391
e-Pares vulnerable to cross-site request forgery

Overview

e-Pares contains a cross-site request forgery vulnerability.

Products Affected

  • e-Pares V01 L01, L03, L10, L20, L30, L40

Description

e-Pares is a system that manages facility (conference rooms, etc.) information. e-Pares contains a cross-site request forgery vulnerability.

Impact

If a user views a malicious page while logged into e-Pares, facility reservation data may be altered.

Solution

Update the Software
Update to the latest version according to the information provided by the developer.
