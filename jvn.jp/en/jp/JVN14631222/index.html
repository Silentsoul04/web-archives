JVN#14631222
Cybozu Garoon fails to restrict access permissions

Overview

Cybozu Garoon contains multiple vulnerabilities in restrcting access permissions.

Products Affected

  • Cybozu Garoon 3.0.0 to 4.2.2

Description

Cybozu Garoon provided by Cybozu,Inc. is a groupware. Cybozu Garoon contains following multiple vulnerabilities in restricting access permissions.

  • Access restriction flaw in the RSS settings - CVE-2016-4908
    CVSS v3 CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:N/I:L/A:N Base Score: 4.3
    CVSS v2 AV:N/AC:L/Au:S/C:N/I:P/A:N Base Score: 4.0
  • Access restriction flaw in MultiReport filters - CVE-2016-4910
    CVSS v3 CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:N/I:L/A:N Base Score: 4.3
    CVSS v2 AV:N/AC:L/Au:S/C:N/I:P/A:N Base Score: 4.0
  • Access restriction flaw in To-Dos of Space function - CVE-2016-7801
    CVSS v3 CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:N/I:L/A:N Base Score: 4.3
    CVSS v2 AV:N/AC:L/Au:S/C:N/I:P/A:N Base Score: 4.0

Impact

A user may alter or delete other users' private RSS settings. (CVE-2016-4908)
An operational administrator may delete other operational administrators' MultiReport filters. (CVE-2016-4910)
A user may delete other users' To-Dos. (CVE-2016-7801)

Solution

Update the Software
Update to the latest version according to the information provided by the developer.
