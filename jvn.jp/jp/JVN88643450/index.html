Sleipnir Mobile for Android における WebView クラスに関する脆弱性

Sleipnir Mobile for Android には、WebView クラスに関する脆弱性が存在します。

  • Sleipnir Mobile for Android 2.1.0 より前のバージョン
  • Sleipnir Mobile for Android Black Edition 2.1.0 より前のバージョン

Sleipnir Mobile for Android は、Android 向けのウェブブラウザです。Sleipnir Mobile for Android には、WebView クラスに関する脆弱性が存在します。

ユーザが、不正な他の Android アプリケーションを使用した場合、当該製品のデータ領域にある情報が漏えいする可能性があります。

アップデートする
開発者の提供する情報をもとに、最新版へアップデートしてください。
