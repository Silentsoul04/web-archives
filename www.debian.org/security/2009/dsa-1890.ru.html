Рекомендация Debian по безопасности

DSA-1890-1 wxwindows2.4 wxwidgets2.6 wxwidgets2.8 -- переполнение целых чисел

Дата сообщения:
19.09.2009
Затронутые пакеты:
wxwindows2.4
wxwidgets2.6
wxwidgets2.8
Уязвим:
Да
Ссылки на базы данных по безопасности:
В каталоге Mitre CVE: CVE-2009-2369.
Более подробная информация:

Тиляй Ванг обнаружил переполнение целых чисел в wxWidgets, межплатформенном наборе инструментов wxWidgets для C++ для реализации графического интерфейса, которое позволяет выполнять произвольный код при помощи специально сформированного файла в формате JPEG.

В предыдущем стабильном выпуске (etch) эта проблема была исправлена в версии 2.4.5.1.1+etch1 для пакета wxwindows2.4 и в версии 2.6.3.2.1.5+etch1 для пакета wxwidgets2.6.

В стабильном выпуске (lenny) эта проблема была исправлена в версии 2.6.3.2.2-3+lenny1 для пакета wxwidgets2.6 и в версии 2.8.7.1-1.1+lenny1 для пакета wxwidgets2.8.

В тестируемом выпуске (squeeze) эта проблема будет исправлена позже.

В нестабильном выпуске (sid) эта проблема была исправлена в версии 2.8.7.1-2 для пакета wxwidgets2.8 и будет исправлена позже для пакета wxwidgets2.6.

Рекомендуется обновить пакеты wxwidgets.
