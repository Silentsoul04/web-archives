DSA-4833-1 gst-plugins-bad1.0 -- Mise à jour de sécurité

Date du rapport :
18 janvier 2021
Paquets concernés :
gst-plugins-bad1.0
Vulnérabilité :
Oui
Références dans la base de données de sécurité :
Dans le dictionnaire CVE du Mitre : CVE-2021-3185.
Plus de précisions :

Andrew Wesie a découvert un dépassement de tampon dans la prise en charge de H264 de l'infrastructure multimédia GStreamer qui pourrait éventuellement avoir pour conséquence l'exécution de code arbitraire.

Pour la distribution stable (Buster), ce problème a été corrigé dans la version 1.14.4-1deb10u1.

Nous vous recommandons de mettre à jour vos paquets gst-plugins-bad1.0.

Pour disposer d'un état détaillé sur la sécurité de gst-plugins-bad1.0, veuillez consulter sa page de suivi de sécurité à l'adresse : https://security-tracker.debian.org/tracker/gst-plugins-bad1.0.
