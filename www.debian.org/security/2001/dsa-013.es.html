Aviso de seguridad de Debian

DSA-013 MySQL -- desbordamiento de búfer remoto

Fecha del informe:
23 de ene de 2001
Paquetes afectados:
mysql
Vulnerable:
Sí
Referencias a bases de datos de seguridad:
En la base de datos de Bugtraq (en SecurityFocus): Id. en BugTraq 2262.
En el diccionario CVE de Mitre: CVE-2001-1274.
Información adicional:
Nicolas Gregoire ha informado de un desbordamiento de búfer en el servidor de mysql que conduce a una explotación remota. Un atacante podía ganar los privilegios de mysqld (y por tanto ganar el acceso a todas las bases de datos).

Le recomendamos que actualice su paquete mysql inmediatamente.

Arreglado en:

Debian 2.2 (potato)
