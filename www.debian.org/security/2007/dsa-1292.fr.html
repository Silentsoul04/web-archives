Bulletin d'alerte Debian

DSA-1292-1 qt4-x11 -- Mauvaise vérification des entrées

Date du rapport :
15 mai 2007
Paquets concernés :
qt4-x11
Vulnérabilité :
Oui
Références dans la base de données de sécurité :
Dans le système de suivi des bogues Debian : Bogue 417391.
Dans la base de données de suivi des bogues (chez SecurityFocus) : Identifiant BugTraq 23269.
Dans le dictionnaire CVE du Mitre : CVE-2007-0242.
Plus de précisions :

Andreas Nolden a découvert un bogue dans les routines de décodage UTF-8 de qt4-x11, une bibliothèque C++ de construction d'interfaces graphiques. Ce bogue pouvait permettre à des utilisateurs distants de conduire des attaques par script intersites (« cross-site scripting » ou « XSS ») et de traversée de répertoires, en utilisant des séquences longues se décodant en de dangereux métacaractères.

Pour l'actuelle distribution stable (Etch), ce problème a été corrigé dans la version 4.2.1-2etch1.

Pour les distributions de test (Lenny) et instable (Sid), ce problème a été corrigé dans la version 4.2.2-2.

Nous vous recommandons de mettre à jour vos paquets qt4-x11.

Corrigé dans :

Debian GNU/Linux 4.0 (etch)
