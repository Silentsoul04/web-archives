Debian-Sicherheitsankündigung

DSA-1246-1 openoffice.org -- Pufferüberlauf

Datum des Berichts:
08. Jan 2007
Betroffene Pakete:
openoffice.org
Verwundbar:
Ja
Sicherheitsdatenbanken-Referenzen:
In der Debian-Fehlerdatenbank: Fehler 405679, Fehler 405986.
In Mitres CVE-Verzeichnis: CVE-2006-5870.
Weitere Informationen:

John Heasman von Next Generation Security Software entdeckte einen Heap-Überlauf im Umgang mit Windows Metafiles in OpenOffice.org, der freien Office-Suite. Dies kann zu einer Diensteverweigerung (denial of service) und möglicherweise zur Ausführung beliebigen Codes führen.

Für die Stable-Distribution (Sarge) wurde dieses Problem in Version 1.1.3-9sarge4 behoben.

Für die Unstable-Distribution (Sid) wurde dieses Problem in Version 2.0.4-1 behoben.

Wir empfehlen Ihnen, Ihr openoffice.org-Paket zu aktualisieren.

Behoben in:

Debian GNU/Linux 3.1 (sarge)
