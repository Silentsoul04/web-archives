DSA-1270-2 openoffice.org -- несколько уязвимостей

Дата сообщения:
20.03.2007
Затронутые пакеты:
openoffice.org
Уязвим:
Да
Ссылки на базы данных по безопасности:
В каталоге Mitre CVE: CVE-2007-0002, CVE-2007-0238, CVE-2007-0239.
Более подробная информация:

В OpenOffice.org, свободном наборе программ для офисной работы, было обнаружено несколько проблем, связанных с безопасностью. Проект Common Vulnerabilities and Exposures определяет следующие проблемы:

  • CVE-2007-0002

    Сотрудники iDefense сообщили о нескольких переполнениях целых чисел в libwpd, библиотеке для обработки документов в формате WordPerfect, которая входит в состав OpenOffice.org. Злоумышленники могут использовать эти уязвимости при помощи специально сформированных файлов WordPerfect, что вызывает аварийную остановку приложения, скомпанованого с libwpd, или возможное выполнение произвольного кода.

  • CVE-2007-0238

    Сотрудники Next Generation Security обнаружили, что код для грамматического разбора из StarCalc в OpenOffice.org содержит переполнение стека, которое может использоваться специально сформированным документом для выполнения произвольного кода.

  • CVE-2007-0239

    Было сообщено, что OpenOffice.org не выполняет экранирование метасимволов командной оболочки, а потому уязвим к выполнению произвольных команд командной оболочки с помощью специально сформированного документа, в котором пользователь нажимает на подготовленную ссылку.

Данное обновление рекомендации предоставляет лишь пакеты для готовящегося выпуска etch, Debian GNU/Linux 4.0.

В стабильном выпуске (sarge) эти проблемы были исправлены в версии 1.1.3-9sarge6.

В тестируемом выпуске (etch) эти проблемы были исправлены в версии 2.0.4.dfsg.2-5etch1.

В нестабильном выпуске (sid) эти проблемы были исправлены в версии 2.0.4.dfsg.2-6.

Рекомендуется обновить пакеты OpenOffice.org.

Исправлено в:

Debian GNU/Linux 3.1 (sarge)
