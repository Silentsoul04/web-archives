Debian 安全报告

DSA-200-1 samba -- 远端安全漏洞 (remote exploit)

报告日期:
2002/11/22
受影响的软件:
samba
可被袭击:
是
参考的安全性数据库:
在 Mitre's CVE 的目录中: CVE-2002-1318.
更详尽的信息:

Steve Langasek 发现一个在 samba 密码处理上的安全漏洞：当它要将 DOS code-page 转变为 little endian UCS2 unicode 时，有一个暂存器的长度未做检查，而可能导致暂存器溢位。目前还没有接收到任何因此而被入侵的消息，但 强烈建议大家尽速更新。

这个问题在 Debian 中的 samba 软件包 2.2.3a-12 版及 2.2.7 版中已做修正。
