För den stabila utgåvan (Jessie) har dessa problem rättats i version 47.0.2526.73-1~deb8u1.

För uttestningsutgåvan (Stretch) kommer dessa problem att rättas inom kort.

För den instabila utgåvan (Sid) har dessa problem rättats i version 47.0.2526.73-1.

Vi rekommenderar att ni uppgraderar era chromium-browser-paket.
