Aviso de seguridad de Debian

mailx -- explotación local

Fecha del informe:
8 de ago de 2000
Paquetes afectados:
mailx
Vulnerable:
Sí
Referencias a bases de datos de seguridad:
En el diccionario CVE de Mitre: CVE-2000-0703.
Información adicional:
mailx es usado frecuentemente por otros programas para enviar correo. Desafortunadamente, mailx tal y como se distribuía con Debian GNU/Linux 2.1 (slink) y Debian GNU/Linux 2.2 (potato) tiene algunas características que hacían posible ejecutar comandos del sistema si un usuario puede engañar a un programa privilegiado para enviar correo usando /usr/bin/mail.

Esto ha sido reparado en la versión 8.1.1-10.1.1slink.2 (para Debian GNU/Linux 2.1) y en la versión 8.1.1-10.1.3 (para Debian GNU/Linux 2.2) no permitiendo más que todas las opciones de configuración se fijen usando el entorno.
