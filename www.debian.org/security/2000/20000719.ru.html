Рекомендация Debian по безопасности

ftpd -- переполнение буфера в ftpd

Дата сообщения:
11.07.2000
Затронутые пакеты:
netstd (ftpd)
Уязвим:
Да
Ссылки на базы данных по безопасности:
На данный момент ссылки на внешние базы данных по безопасности отсутствуют.
Более подробная информация:
Служба ftpd из состава пакет netstd из Debian GNU/Linux 2.1 (slink) уязвима к широко обсуждаемой "ошибке setproctitle". Служба ftpd в ещё не выпущенном Debian GNU/Linux 2.2 (potato) также уязвима.

Эта проблема была исправлена в netstd 3.07-7slink.4 для Debian GNU/Linux 2.1 (slink) и в ftpd 0.11-8potato.1 для Debian GNU/Linux 2.2 (potato). Рекомендуется как можно скорее обновить пакет ftpd.
