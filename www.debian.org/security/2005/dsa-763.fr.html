Bulletin d'alerte Debian

DSA-763-1 zlib -- Déni de service distant

Date du rapport :
20 juillet 2005
Paquets concernés :
zlib
Vulnérabilité :
Oui
Références dans la base de données de sécurité :
Dans le dictionnaire CVE du Mitre : CVE-2005-1849.
Plus de précisions :

Markus Oberhumer a découvert une faille dans la gestion des entrées invalides par zlib, une bibliothèque utilisée pour la compression et la décompression de fichiers. L'ouverture d'un fichier invalide peut provoquer le plantage des programmes utilisant zlib.

L'ancienne distribution stable (Woody) n'est pas touchée par cette vulnérabilité.

Pour l'actuelle distribution stable (Sarge), ce problème a été corrigé dans la version 1.2.2-4.sarge.2.

Pour la distribution instable (Sid), ce problème a été corrigé dans la version 1.2.3-1.

Nous vous recommandons de mettre à jour votre paquet zlib.

Corrigé dans :

Debian GNU/Linux 3.1 (stable)
