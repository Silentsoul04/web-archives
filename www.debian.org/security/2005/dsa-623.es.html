Aviso de seguridad de Debian

DSA-623-1 nasm -- desbordamiento de buffer

Fecha del informe:
4 de ene de 2005
Paquetes afectados:
nasm
Vulnerable:
Sí
Referencias a bases de datos de seguridad:
En el sistema de seguimiento de errores de Debian: error 285889.
En el diccionario CVE de Mitre: CVE-2004-1287.
Información adicional:

Jonathan Rockway descubrió un desbordamiento de buffer en nasm, el ensamblador de propósito general para x86, que podría conducir a la ejecución de código arbitrario cuando se compilara un archivo pernicioso fuente en ensamblador.

Para la distribución estable (woody), este problema se ha corregido en la versión 0.98.28cvs-1woody2.

Para la distribución inestable (sid), este problema se ha corregido en la versión 0.98.38-1.1.

Le recomendamos que actualice el paquete nasm.
