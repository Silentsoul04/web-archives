Alerta de Segurança Debian

DSA-623-1 nasm -- estouro de buffer

Data do Alerta:
04 Jan 2005
Pacotes Afetados:
nasm
Vulnerável:
Sim
Referência à base de dados de segurança:
No sistema de acompanhamento de bugs do Debian: Bug 285889.
No dicionário CVE do Mitre: CVE-2004-1287.
Informações adicionais:

Jonathan Rockway descobriu um estouro de buffer no nasm, o assembler x86 de propósito geral, que poderia levar à execução de código arbitrário durante a compilação de um arquivo fonte assembly malicioso.

Na distribuição estável (woody), este problema foi corrigido na versão 0.98.28cvs-1woody2.

Na distribuição instável (sid), este problema foi corrigido na versão 0.98.38-1.1.

Nós recomendamos que você atualize seu pacote nasm.

Corrigido em:

Debian GNU/Linux 3.0 (woody)
