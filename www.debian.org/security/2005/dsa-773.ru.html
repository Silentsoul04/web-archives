DSA-773-1 amd64 -- несколько уязвимостей

Дата сообщения:
11.08.2005
Затронутые пакеты:
several
Уязвим:
Да
Ссылки на базы данных по безопасности:
На данный момент ссылки на внешние базы данных по безопасности отсутствуют.
Более подробная информация:

Данная рекомендация добавляет поддержку безопасности для стабильного выпуска архитектуры amd64. Поддержка касается всех обновлений безопасности с момента выпуска sarge, которые не были обновлены когда перенос на архитектуру amd64 ещё не был официальным. В будущем рекомендации безопасности будут включать в себя обновления для указанного переноса.
