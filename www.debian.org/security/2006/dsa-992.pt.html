Alerta de Segurança Debian

DSA-992-1 ffmpeg -- estouros de pilha

Data do Alerta:
10 Mar 2006
Pacotes Afetados:
ffmpeg
Vulnerável:
Sim
Referência à base de dados de segurança:
No sistema de acompanhamento de bugs do Debian: Bug 342207.
No dicionário CVE do Mitre: CVE-2005-4048.
Informações adicionais:

Simon Kilvington descobriu que imagens PNG especialmente preparadas podem disparar um estouro de "heap" no libavcodec, a biblioteca multimídia do ffmpeg, que pode levar à execução de código arbitrário.

A antiga distribuição estável ("woody") não contém o pacote ffmpeg.

Para a distribuição estável ("sarge") este problema foi corrigido na versão 0.cvs20050313-2sarge1.

Para a distribuição instável ("sid") este problema foi corrigido na versão 0.cvs20050918-5.1.

Recomendamos que você atualize seu pacote ffmpeg.

Corrigido em:

Debian GNU/Linux 3.1 (sarge)
