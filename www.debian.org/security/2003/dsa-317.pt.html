Alerta de Segurança Debian

DSA-317-1 cupsys -- negação de serviço

Data do Alerta:
11 Jun 2003
Pacotes Afetados:
cupsys
Vulnerável:
Sim
Referência à base de dados de segurança:
Na base de dados do BugTraq (na SecurityFocus): ID BugTraq 7637.
No dicionário CVE do Mitre: CVE-2003-0195.
Informações adicionais:

O servidor de impressão CUPS no Debian é vulnerável a um atacante de negação de serviço quando uma requisição HTTP é recebida sem ter sido adequadamente terminada.

Na atual distribuição estável (woody), este problema foi corrigido na versão 1.1.14-5.

Na antiga distribuição estável (potato), este problema foi corrigido na versão 1.0.4-12.2.

Na distribuição instável (sid), este problema foi corrigido na versão 1.1.19final-1.

Nós recomendamos que você atualize seus pacotes cupsys.
