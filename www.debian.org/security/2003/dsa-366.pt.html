Alerta de Segurança Debian

DSA-366-1 eroaster -- arquivo temporário inseguro

Data do Alerta:
05 Ago 2003
Pacotes Afetados:
eroaster
Vulnerável:
Sim
Referência à base de dados de segurança:
Na base de dados do BugTraq (na SecurityFocus): ID BugTraq 8350.
No dicionário CVE do Mitre: CVE-2003-0656.
Informações adicionais:

eroaster, uma interface para a queima de CD-Rs usando o cdrecord, não toma as devidas precauções de segurança ao criar um arquivo temporário para ser usado como lockfile. Esta falha pode potencialmente ser explorada para sobrescrever arquivos arbitrários com os privilégios do usuário que está executando o eroaster.

Na atual distribuição estável (woody) este problema foi corrigido na versão 2.1.0.0.3-2woody1.

Na distribuição instável (sid) este problema foi corrigido na versão 2.2.0-0.5-1.

Nós recomendamos que você atualize seus pacotes eroaster.

Corrigido em:

Debian GNU/Linux 3.0 (woody)
