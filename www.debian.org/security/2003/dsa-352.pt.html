Alerta de Segurança Debian

DSA-352-1 fdclone -- diretório temporário inseguro

Data do Alerta:
22 Jul 2003
Pacotes Afetados:
fdclone
Vulnerável:
Sim
Referência à base de dados de segurança:
Na base de dados do BugTraq (na SecurityFocus): ID BugTraq 8247.
No dicionário CVE do Mitre: CVE-2003-0596.
Informações adicionais:

O fdclone cria um diretório temporário no /tmp para trabalhar nele. No entanto, se o diretório já existe, o diretório existente é utilizado, desconsiderando seu dono ou permissões. Isto pode levar um atacante a obter acesso aos arquivos temporários do fdclone e o conteúdo dos mesmos ou a substituí-los por outros arquivos que o atacante desejar.

Na atual distribuição estável (woody) este problema foi corrigido na versão 2.00a-1woody3.

Na antiga distribuição estável (potato) este problema foi corrigido na versão 2.04-1.

Nós recomendamos que você atualize seus pacotes fdclone.

Debian GNU/Linux 3.0 (woody)
