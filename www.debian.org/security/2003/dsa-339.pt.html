Alerta de Segurança Debian

DSA-339-1 semi -- arquivo temporário inseguro

Data do Alerta:
06 Jul 2003
Pacotes Afetados:
semi, wemi
Vulnerável:
Sim
Referência à base de dados de segurança:
Na base de dados do BugTraq (na SecurityFocus): ID BugTraq 8115.
No dicionário CVE do Mitre: CVE-2003-0440.
Informações adicionais:

NOTA: devido a uma combinação de problemas administrativos, este alerta foi erroneamente enviado anteriormente com o identificador "DSA-337-1". O DSA-337-1 refere-se a um alerta sobre o gtksee.

semi, uma biblioteca MIME para o GNU Emacs, não toma as devidas precauções de segurança ao criar arquivos temporários. Esta falha pode potencialmente ser explorada para sobrescrever arquivos arbitrários com os privilégios do usuário que está executando o Emacs e a semi, provavelmente com o conteúdo fornecido pelo atacante.

wemi é um fork da semi e contém a mesma falha.

Na atual distribuição estável (woody) este problema foi corrigido na versão 1.14.3.cvs.2001.08.10-1woody2 da semi e na versão 1.14.0.20010802wemiko-1.3 da wemi.

Na distribuição instável (sid) este problema foi corrigido na versão 1.14.5+20030609-1 da semi. A distribuição instável não contém o pacote wemi.

Nós recomendamos que você atualize seus pacotes semi e wemi.

Corrigido em:

Debian GNU/Linux 3.0 (woody)
