Рекомендация Debian по безопасности

DSA-2938-1 -- Доступность долгосрочной поддержки для Debian 6.0 / squeeze

Дата сообщения:
27.05.2014
Затронутые пакеты:
Уязвим:
Нет
Ссылки на базы данных по безопасности:
На данный момент ссылки на внешние базы данных по безопасности отсутствуют.
Более подробная информация:

Завершена начальная организация и настройка Squeeze LTS, долгосрочная поддержка готова в плане предоставления поддержки безопасности и начнётся в конце этого месяца, когда прекратится обычная поддержка:

Информация для пользователей

Поддержка Squeeze LTS продлится в течении пяти лет с момента выпуска Squeeze, т. е. до 6 февраля 2016 года.

Вам следует вручную добавить источники apt для squeeze-lts. Информацию о том, как это сделать, вы найдёте по адресу https://wiki.debian.org/LTS/Development#Add_squeeze-lts_to_your_sources.list

Вам следует подписаться на новый список рассылки, который будет использоваться для анонсов обновлений безопасности для squeeze-lts: https://lists.debian.org/debian-lts-announce/

Поддержка Squeeze LTS не покрывает некоторые пакеты. Они могут быть обнаружены при помощи нового инструмента, debian-security-support. Информация о том, как его использовать, может быть найдена здесь: https://wiki.debian.org/LTS/Development#Check_for_unsupported_packages

Если debian-security-support обнаруживает пакет, который более не поддерживается, но всё равно важен для вас, свяжитесь с debian-lts@lists.debian.org (см. ниже).

Архив squeeze-backports будет поддерживаться на протяжении жизненного цикла Squeeze LTS.

Информация для сопровождающих Debian

Во-первых, сопровождающие пакетов Debian не обязаны работать над обновлениями своих пакетов для squeeze-lts. Обновления пакетов для squeeze-lts будут предоставляться командой Debian LTS.

Тем не менее, если вы действительно заинтересованы в этом (а сопровождающий всегда знает свой пакет лучше, чем другие люди), мы с радостью примем вашу помощь; всякий с Debian.org, а также тот, чей ключ добавлен в брелок сопровождающих Debian, может осуществлять загрузку в squeeze-lts. Информация о том, как осуществлять загрузку исправленного пакета, может быть найдена по адресу https://wiki.debian.org/LTS/Development#Upload_Packages

Списки рассылки

Координация Debian LTS осуществляется через список рассылки debian-lts: https://lists.debian.org/debian-lts/

Подпишитесь на него или добавьте этот список к себе через GMANE (gmane.linux.debian.devel.lts)

Помимо списка debian-lts-announce существует ещё список рассылки для анонса всех загрузок в debian-lts: https://lists.debian.org/debian-lts-changes/

Отслеживание безопасности

Вся информация о статусе различных уязвимостей (напр., если версия в squeeze-lts не подвержена какой-то уязвимости, хотя версия в wheezy подвержена ей) будет отслеживаться в системе отслеживания безопасности Debian:

https://security-tracker.debian.org

Если вы заметите ошибку в данных, см. https://security-tracker.debian.org/tracker/data/report
